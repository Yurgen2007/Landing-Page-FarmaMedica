# FarmaMedica - Droguería

## Instalación
Clona el repositorio con el siguiente comando 
```bash
$ git clone https://github.com/Yurgen2007/LandingPage-FarmaMedical.git
```

## Instalación

Abre la carpeta donde deseas ejecutar el proyecto y simplemente haz doble clic en el archivo `index.html` para abrirlo en tu navegador.

```bash
$ start index.html
```

## Servidor Local (Opcional)

Si prefieres usar un servidor local para una mejor experiencia (evita problemas con rutas relativas), puedes usar una de las siguientes opciones:

### Opción 1: Live Server (VSCode)

1. Abre el proyecto en VSCode
2. Instala la extensión "Live Server"
3. Haz clic derecho en `index.html` y selecciona "Open with Live Server"

### Opción 2: Python

```bash
$ python -m http.server 8000
```

Luego abre tu navegador en `http://localhost:8000`

### Opción 3: Node.js

```bash
$ npx serve
```

## Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- No requiere instalación de dependencias adicionales
- No requiere base de datos
- Es un proyecto frontend estático
